# 라즈베리파이를 활용한 미니 프로젝트


## 제목 : 불면증극복 

## 작품 개요 :
현대인의 불면증을 해결 할 수 있는 오르골을 만들고자 함 
실행을 시키면 1번트랙과 2번트랙에 있는 노래들이 반복적으로 재생된다. 


## 작품 제작에 사용한 센서와 엑추에이터와 역할

부저 : 소리를 내는 모듈
초음파센서 : 초음파를 발사하여 반사되어 오는 반사 신호를 받을 때까지의 시간을 확인하여 거리를 측정하는 센서
